Список переписок

Требуется отображать список всех переписок всех девушек данного агенства.
Список всех переписок. Каждая девушка в отдельности может общаться с N количеством Мужчин. В данном случае требуется вывести Список всех девушек текущего Агенства с указанием следующих параметров:
1. Lady's ID. ID девушки на сайте
2. Lady's name. ИФ девушки. Является ссылкой на страницу со списком всех переписок девушки (см. 13 строку в этом excel)
3. Conversations amount. Кол-во переписок конкретной девушки. Каждая девушка в отдельности может общаться с N количеством Мужчин. вот это N и нужно вывести
4. Last incoming letter. Дата и время последнего входящего сообщения (письма). И не важно с каким именно мужчиной.
5. Last outgoing letter. Дата и время последнего исходящего сообщения (письма). И не важно с каким именно мужчиной.
6. Go to converstions. Является ссылкой на страницу со списком всех переписок девушки (см. 13 строку в этом excel). Отображать ввиде текста View Conversations"

-------------------------------------------
Сортировка списка

1. По умолчанию: сортировка по дате последнего сообщения (письма) - Last incoming letter.
2. Сверху в низ: От самого последнего до самого позднего.
3. Сортировка возможна для Last incoming letter и Last outgoing letter
4. Хотелось бы чтобы активная сортировка выделялась цветом ( ну хотя бы заголовок столбца)"

Динамическая подгрузка переписок

Список переписок отображается без пагинации. По умолчанию 20 первых переписок. Сделать в конце списка кнопки ""Show Next 20""  и ""Shaw All"". Формат: стандартные из бутстрапа
1. Show Next 20. Кнопка активна только в случае если есть ещё переписки для отображения. Если уже отображены все доступные переписки - сделать кнопку неактивной.
2. Show Next 40. Кнопка активна только в случае если есть ещё переписки для отображения. Если уже отображены все доступные переписки - сделать кнопку неактивной."

--------------------------------------------
Динамическая подгрузка сообщений: вывод кол-ва показанных переписок

1. Отображать кол-во отображенных переписок на странице. Формат просто там 60 из 120 , или 27 из 27
2. вывести сверху списка где то возле фильтров"
Кнопка Show Next 20"  По нажатию подгружается следующие переписки (max 20 шт). Кнопка становится неактивной если больше нет данных для отображения

--------------------------------------------
Кнопка "Show Next 40"
По нажатию подгружается все доступные переписки. Кнопки Show next 20 и Show all становятся неактивными

------------------------------------------
Фильтр "Sort by Letter Type"

Фильтр для отображения переписок по отдельным признакам
1. All conversations (значение по умолчанию)
2. With Unread letters. Этот статус касается только не прочитанных девушкой, т.е. входящие письма для девушек. Сохраняется принцип стандартной сортировки (см. выше Сортировка списка) - сверху списка отображать переписки в которых дата непрочитанного письма самая раняя (при чем надо учесть что непрочитанных писем может быть больше 1)"

------------------------------------
Фильтр "Sort by Time"

Фильтр для отображения переписок за отдельные периоды:
1. All Time (значение по умолчанию)
2. Today
3. On this Week
4. On this Month
Важно - принцип базовой сортировки по дате/времени сохраняется"

----------------------------------
Кнопка Clear

По нажатию обнуляются все заданные ранее фильтры (устанавливаются в значения по умолчанию) и отображаются первые 20 переписок

Mail (главная) - Переписки конкретной девушки - Конкретная переписка с конкретным мужчиной"
